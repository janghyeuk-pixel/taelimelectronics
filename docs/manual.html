🔐
로그인 방법
사이트에 처음 접속할 때 비밀번호를 입력합니다
1
사이트 접속
인터넷 브라우저(크롬 권장)를 열고 주소창에 사이트 주소를 입력합니다. 즐겨찾기에 저장해 두시면 편합니다.
2
비밀번호 입력
화면 가운데 비밀번호 칸에 숫자/영문을 입력합니다. ▸ 대표님 비밀번호: admin2024 ▸ 직원 비밀번호: taelim2024 (비밀번호는 대소문자를 구분합니다. Caps Lock이 켜져 있으면 안 됩니다.)
3
로그인 버튼 클릭
비밀번호 입력 후 파란 [로그인] 버튼을 클릭합니다. 잠시 기다리면 메인 화면으로 이동합니다.
4
로그아웃
오른쪽 상단 [로그아웃] 버튼을 누르면 안전하게 종료됩니다. 공용 PC에서는 반드시 로그아웃 해주세요.
5
비밀번호를 잊어버렸을 때
설정 탭에서 비밀번호를 변경할 수 있습니다. 접속이 안 될 경우 박장혁 이사에게 연락 주세요.
💡 로그인 화면 하단 [이메일로 로그인] 버튼은 직원 개인 계정용입니다. 대표님은 위 비밀번호로 로그인하시면 됩니다.
⚡
검침 입력
매월 전기·수도 계량기 숫자를 기록합니다
1
검침 입력 탭 클릭
상단 메뉴에서 [검침 입력]을 클릭합니다.
2
적용 기간 입력
이번 달 검침 기간을 입력합니다. 예) 시작일: 2025-05-08 / 종료일: 2025-06-07
3
전기 검침값 입력
각 층별로 [전월] 숫자와 [금월] 숫자를 입력합니다. ▸ 전월: 지난달 계량기 숫자 ▸ 금월: 이번달 계량기 숫자
4
수도 검침값 입력
전기와 동일하게 각 층별로 전월/금월 수도 계량기 숫자를 입력합니다.
5
수도 요금 청구 여부
수도 요금을 이번 달에 청구할지 선택합니다. ▸ O: 이번 달 수도 요금 청구함 ▸ X: 이번 달 수도 요금 청구 안 함 (격월 청구 시 사용)
6
전기·수도 고지서 금액 입력
한전에서 온 전기 고지서와 수도 고지서의 금액을 입력합니다.
7
히스토리 저장
모든 입력이 끝나면 하단 [💾 히스토리 저장] 버튼을 클릭합니다.
💡 숫자를 잘못 입력했을 경우 언제든지 다시 수정하고 저장할 수 있습니다.
📋
청구서 발행
임차인에게 관리비 청구서를 출력하거나 이메일로 보냅니다
1
청구서 탭 클릭
상단 메뉴에서 [청구서]를 클릭합니다.
2
업체 선택
화면 상단에서 청구서를 발행할 업체를 선택합니다. ▸ 1층 웨지우드 / 2층 태하무역 / 3층 유연어패럴
3
청구 내용 확인
자동으로 계산된 청구 내용이 표시됩니다. 임대료, 관리비, 전기세, 수도세, 부가세 등을 확인합니다.
4
PDF 출력
[🎨 컬러 PDF 출력] 또는 [⬜ 흑백 PDF 출력] 버튼을 클릭합니다. PDF로 저장하려면 프린터 대신 "PDF로 저장"을 선택합니다.
5
이메일 발송 (선택)
[📧 이메일 발송] 버튼을 클릭하면 Gmail이 열리면서 청구서가 첨부됩니다.
💡 청구서는 매월 검침 후 발행합니다. 발행 전에 검침 입력이 저장되어 있어야 정확한 금액이 계산됩니다.
🚨
긴급 호출
응급 상황 시 대표님께 즉시 Telegram 알림을 보냅니다
1
전자결재 탭 클릭
상단 메뉴에서 [전자결재]를 클릭합니다.
2
긴급 호출 버튼 클릭
빨간 패널 안의 [🚨 긴급 호출] 버튼을 클릭합니다.
3
확인 후 전송
확인 창에서 [확인 — 전송] 버튼을 클릭하면 대표님 Telegram으로 즉시 알림이 전송됩니다.
4
미확인 자동 재전송
5분 후에도 확인이 없으면 자동으로 다시 전송됩니다.
⚠️ 긴급 호출은 실제 응급 상황에만 사용해 주세요.
📄
전표 작성
입금·출금·대체 전표를 작성하고 PDF로 출력합니다
1
전표 탭 클릭
상단 메뉴에서 [전표]를 클릭합니다.
2
전표 종류 선택
▸ 입금전표: 돈이 들어올 때 (임대료 수령 등) ▸ 출금전표: 돈이 나갈 때 (비용 지불 등) ▸ 대체전표: 계좌 간 이체, 비용 배분 등
3
내용 입력
날짜, 금액, 계정과목, 거래처, 적요를 입력합니다.
4
전표 저장 및 출력
[💾 전표 저장] 후 목록에서 [출력] 버튼을 클릭하면 A4 PDF가 생성됩니다.
💡 색깔 구분: 빨간줄=입금, 파란줄=출금, 검은줄=대체
📅
출퇴근 체크
출근·퇴근 시간을 기록하고 조회합니다
1
출퇴근 탭 클릭
상단 메뉴에서 [출퇴근]을 클릭합니다.
2
출근 체크
출근 시 [✅ 출 근] 버튼을 클릭합니다. 현재 시각이 자동으로 기록됩니다.
3
퇴근 체크
퇴근 시 [🏃 퇴 근] 버튼을 클릭합니다. 근무 시간이 자동으로 계산됩니다.
4
기록 조회
[기록 조회] 탭에서 날짜별, 월별 출퇴근 내역을 확인할 수 있습니다.
💡 출근 버튼은 하루에 한 번만 눌러주세요.
🔥
비상연락망 (소방)
소방 담당자 연락처를 관리하고 긴급 시 바로 연락합니다
1
전자결재 탭 → 비상연락망 탭
상단 메뉴에서 [전자결재]를 클릭 후, 화면 안의 [🔥 비상연락망] 탭을 클릭합니다.
2
연락처 확인 및 전화
소방서, 경찰서, 가스회사 등의 연락처를 확인합니다. 전화번호 클릭 시 바로 전화 앱이 열립니다.
3
소방 문서 보관
소방계획서, 점검 기록 등은 [📁 소방·안전 문서 보관함]에 사진으로 업로드하여 보관합니다.
⚠️ 119 신고는 직접 119에 전화하세요. 비상연락망은 건물 관계자 연락용입니다.
📑
계약서 관리
임차인별 임대 계약서를 사진으로 보관합니다
1
임차인 현황 탭 클릭
상단 메뉴에서 [임차인 현황]을 클릭합니다.
2
계약서 버튼 클릭
업체 카드 하단의 [📑 계약서] 버튼을 클릭합니다.
3
사진 추가
스마트폰 카메라로 계약서를 촬영하거나 스캔 파일을 선택합니다. 여러 장인 경우 반복 추가합니다.
💡 계약서 원본은 안전한 곳에 별도 보관하고, 이 시스템에는 사진 백업으로 활용하세요.
💰
자금 현황
회사 계좌별 잔액과 자금 흐름을 관리합니다
1
자금현황 탭 클릭
상단 메뉴에서 [자금현황]을 클릭합니다.
2
이월잔액 입력
해당 월 시작 잔액을 이월잔액 칸에 입력합니다. (전월 말 총 잔고)
3
통장 내역 가져오기
기업은행 거래내역 XLS 파일을 [보통018 가져오기] / [보통032 가져오기] 버튼으로 업로드합니다. 자동으로 내역이 추가됩니다.
4
계좌 잔액 수동 확인
예금·잔고 현황 표에서 MMF, 018, 032, 현금 잔액을 실제 통장과 맞춰 클릭 후 수정합니다.
5
인쇄
하단 [🖨️ 자금현황 인쇄] 버튼을 클릭하면 A4 보고서가 생성됩니다.
💡 월 마감 때 통장 잔액을 확인하여 업데이트해 주세요.
📢
업무 보고
일일 업무 내용을 기록하고 공유합니다
1
업무보고 탭 클릭
상단 메뉴에서 [업무보고]를 클릭합니다.
2
새 보고 작성
[✏️ 새 업무보고 작성] 버튼을 클릭합니다.
3
날짜 및 내용 입력
날짜를 선택하고 오늘 한 업무 내용을 입력합니다. 예) "1층 웨지우드 관리비 청구서 발송 완료", "화장실 형광등 교체" 등
4
저장
[저장] 버튼을 클릭합니다. 저장된 보고는 목록에서 날짜별로 확인 가능합니다.
💡 매일 간단하게라도 업무 기록을 남겨두시면 나중에 참고하기 좋습니다.
